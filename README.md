### Descripción.

_Este Conversor de moneda permite cambiar entre seis monedas:_

- USD dólar. 

- EUR euro. 

- COP peso colombiano. 

- GBP Libra esterlina. 

- KRW Won Surcoreano. 

- JPY yen.
