## Descripción.

_Este Conversor de moneda permite cambiar entre seis monedas:_

- USD dólar. 

- EUR euro. 

- COP peso colombiano. 

- GBP Libra esterlina. 

- KRW Won Surcoreano. 

- JPY yen. 

<p>
    Proyecto desarrollado con el fin de implementar los conocimientos adquiridos a lo largo de Java Orientado a Objetos G4 - ONE.
</p>
